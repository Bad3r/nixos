## Hime

Hime is an extremely easy-to-use input method framework. It is lightweight, stable, powerful and supports many commonly used input methods, including Cangjie, Zhuyin, Dayi, Rank, Shrimp, Greek, Korean Pinyin, Latin Alphabet, etcâ€¦

The following snippet can be used to configure Hime:

```programlisting
{
  i18n.inputMethod = {
    enable = true;
    type = "hime";
  };
}
```
